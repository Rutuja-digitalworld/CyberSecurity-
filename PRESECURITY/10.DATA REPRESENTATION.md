# Colors, Bits & Hexadecimal – Quick Notes
### RGB Colors

Computers create colors using RGB (Red, Green, Blue).

Each color can be turned ON (1) or OFF (0). With only ON/OFF states:

## Bit
•A Bit is the smallest unit of data.

•Can have only 0 or 1.

## Byte (Octet)
• 8 Bits = 1 Byte

•1 Byte = 256 possible values

## Hexadecimal (Hex)
• Hexadecimal makes binary easier to read.

• 1 Hex Digit = 4 Bits

### DECIMAL TO HEXADECIMAL 
•Decimal = Base 10 (0-9)

•Binary = Base 2 (0,1)

•Octal = Base 8 (0-7)

•Hex = Base 16 (0-9, A-F)

•1 Bit = 0 or 1

•8 Bits = 1 Byte

•1 Hex Digit = 4 Bits

•1 Octal Digit = 3 Bits

Binary Place Values:
8 4 2 1

**Hex:**

A=10

B=11

C=12

D=13

E=14

F=15
