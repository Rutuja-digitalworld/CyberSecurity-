# 🐧 Linux - Quick Notes
Where is Linux Used?

Linux powers:

🌐 Websites & Web Servers

🚗 Car infotainment/control systems

🛒 Point of Sale (POS) systems (shop tills/registers)

🚦 Critical infrastructure (traffic lights, industrial sensors)

## What is Linux?
•Linux is an open-source operating system based on UNIX.

•It is lightweight, flexible, and widely used.

## Linux Distributions (Distros)
•Linux is an umbrella term for many versions called distributions (distros).

•Different distros are designed for different purposes.

### Common Distros
•Ubuntu → Popular for desktops and servers.

•Debian → Stable and widely used.

📌 Ubuntu Server can run with as little as 512 MB RAM.

**Find Command**

Used to search for files and folders.

---------

**Find a specific file**

find -name passwords.txt → Finds the file named passwords.txt.

------------

**Wildcard (*)**

Represents any characters.

**Find all text files**

find -name "*.txt" → Finds all files ending with .txt.

--------

**Output**

Displays the location/path of matching files (e.g., folder1/passwords.txt).
