# __Operating System (OS)__

•Operating System (OS) is the core software that manages hardware, applications, and users.

•Acts as a bridge between User ↔ Applications ↔ Hardware.

### Examples: Windows, Linux, macOS

### Why Do We Need an OS?

•Manages system resources.

•Prevents conflicts between applications.

•Controls CPU, memory, storage, devices, and security.

•Makes modern computing possible.

### Airport Analogy
Computer	                           Airport

Hardware	        ➡️          Runways, planes, radar

Applications	     ➡️            Airlines & passengers

Operating System	  ➡️               Air Traffic Control System

## OS Responsibilities

✅ Schedules resources

✅ Manages traffic

✅ Resolves conflicts

✅ Ensures system stability
________

# System Privilege Layers
### Kernel Space
•Privileged core of the OS.
•Runs the Kernel.
•Has direct access to:
CPU

Memory


Storage

Hardware

### Purpose

✅ Controls hardware and system resources

----------

### User Space

•Where normal applications run.

•Cannot access hardware directly.

•Must use System Calls to request services from the Kernel.

### Examples
•Browser
•Games
•Music Players

-------

### System Call
•A System Call is a request from an application to the Kernel.

### Used for:
•Opening files
•Playing sounds
•Connecting to Wi-Fi
•Accessing hardware

---------
### Airport Analogy for Privileges
•Kernel Space = Control Tower 🏢
Full control over airport operations.
Direct access to runways and equipment.

------
•User Space = Airlines & Passengers ✈️
Cannot control airport infrastructure directly.
Must request services through the control tower.

### Benefits of Separation

✅ Better security

✅ Better stability

✅ Faulty apps cannot directly damage hardware

✅ One app crashing usually won't crash the entire OS
