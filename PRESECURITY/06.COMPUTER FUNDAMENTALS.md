Boot Process (Startup Sequence)<br>

**The Boot Process is the sequence of steps a computer follows to start and load the Operating System.**<br>

Step 1: Power Button Pressed<br>
Pressing the power button sends a signal to the PSU (Power Supply Unit).<br>
PSU supplies power to all computer components.

Step 2: Firmware Starts (UEFI/BIOS)<br>
UEFI (Unified Extensible Firmware Interface) starts and initializes hardware.<br>
BIOS is the older version of UEFI.<br>
Firmware manages the startup process.<br>
Important<br>
✅ UEFI has mostly replaced BIOS.<br>

Step 3: POST (Power-On Self Test)<br>
UEFI performs a hardware check.<br>
Verifies all required components are:<br>
Present<br>
Properly configured<br>
Functioning correctly<br>
Purpose<br>
✅ Detect hardware problems before booting.<br>

Step 4: Select Boot Device<br>
UEFI checks the boot order list.<br>
Searches for a bootable device.<br>
Examples<br>
SSD,
HDD,
USB Drive,
CD/DVD<br>

Step 5: Bootloader Starts<br>
The Bootloader is loaded from the selected boot device.<br>
It loads the Operating System into RAM.<br>
Control is then handed over to the OS.<br>
Purpose<br>
✅ Loads and starts the Operating System.<br>
