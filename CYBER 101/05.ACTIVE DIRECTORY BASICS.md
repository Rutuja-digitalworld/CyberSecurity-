# **Windows Domain & Active Directory (AD) – Short Notes**

* **Windows Domain:** A group of users and computers managed **centrally** by an organization.
* **Active Directory (AD):** Central repository for managing users, computers, and security policies.
* **Domain Controller (DC):** Server that runs Active Directory services.

**Advantages:**

* **Centralized Identity Management:** Manage all users from one place.
* **Centralized Security Policies:** Apply security settings to all users and computers.

**Real-World Example:**

* School/work login credentials work on any domain computer because authentication is handled by **Active Directory**.
* AD also enforces restrictions (e.g., no Control Panel or admin access).

# **Active Directory Domain Services (AD DS) – Short Notes**

### **AD DS**

* **AD DS (Active Directory Domain Services):** Central directory that stores and manages all network objects (users, groups, computers, printers, etc.).

### **Users**

* **Security Principal:** Can be authenticated and assigned permissions.
* **Types:**

  * **User Accounts:** Represent employees.
  * **Service Accounts:** Used by services (IIS, MSSQL) with limited privileges.

### **Machines (Computers)**

* Every domain-joined computer gets a **Machine Account**.
* Machine accounts are also **Security Principals**.
* **Naming:** `ComputerName$` (e.g., `DC01$`).
* **Password:** Automatically rotated (120 random characters).

### **Security Groups**

* Used to **assign permissions** to users and computers.
* Can contain **users, computers, and other groups**.

### **Important Default Groups**

* **Domain Admins:** Full control over the domain.
* **Server Operators:** Manage Domain Controllers.
* **Backup Operators:** Access all files for backups.
* **Account Operators:** Create/modify user accounts.
* **Domain Users:** All domain users.
* **Domain Computers:** All domain computers.
* **Domain Controllers:** All DCs.

### **Active Directory Users and Computers (ADUC)**

* Tool used to **create, delete, modify users, groups, computers, and reset passwords**.

### **Organizational Units (OUs)**

* Container used to **organize users and computers**.
* Mainly used to **apply Group Policies**.
* A user/computer can belong to **only one OU**.

### **Default AD Containers**

* **Builtin:** Default Windows groups.
* **Computers:** Newly joined computers.
* **Domain Controllers:** Stores DCs.
* **Users:** Default users and groups.
* **Managed Service Accounts:** Service accounts.

### **OU vs Security Group**

| **OU (Organizational Unit)**     | **Security Group**                     |
| -------------------------------- | -------------------------------------- |
| Used to apply **Group Policies** | Used to assign **permissions**         |
| User belongs to **one OU**       | User can belong to **multiple groups** |
| Organizes users/computers        | Controls access to resources           |
