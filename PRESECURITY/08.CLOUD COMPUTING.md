# __Cloud Computing__
Cloud Computing is the delivery of computing resources (servers, storage, databases, networking, etc.) over the Internet.

It allows applications to be accessed from anywhere.

Makes applications more scalable, reliable, and cost-effective.
## Benefits
✅ Access from anywhere

✅ Easy scalability

✅ High availability

✅ Reduced hardware costs

## Evolution of Servers to Cloud

### 1. Physical Servers

One application runs on one physical machine.

Expensive and inefficient.

Resources often remain unused.

### 2. Virtualization

Multiple VMs run on one physical server using a Hypervisor.

Better resource utilization.

### 3. Containers

Lightweight environments that share the host OS kernel.

Faster and more efficient than VMs.

Commonly managed using Docker.

### 4. Cloud Computing

Resources are provided over the Internet.

No need to own physical hardware.

Applications can scale automatically based on demand.

# Cloud Benefits & Characteristics
Benefits

✅ Scalability → Easily increase or decrease resources as needed.

✅ On-Demand Self-Service → Create/remove servers and storage instantly.

✅ Pay-as-You-Go → Pay only for the resources you use.

✅ Security → Cloud providers protect infrastructure with security measures.

✅ High Availability → Applications remain available even if part of the system fails.

✅ Global Access → Applications can be accessed from anywhere in the world.

# Types of Cloud (Deployment Models)

### Public Cloud

•Shared cloud infrastructure provided over the internet.

•Affordable and highly scalable.

•No infrastructure management required.

### Used By
•Startups

•Websites

•Global applications

------
### Private Cloud

•Dedicated cloud environment for a single organization.

•More control, customization, and security.

### Used By
•Banks

•Healthcare organizations

•Government agencies

-----
### __Hybrid Cloud__

•Combination of Public + Private Cloud.

•Sensitive data stays private while other services use the public cloud.

### Used By

•E-commerce companies

•Large enterprises

# __Cloud Service Models__

### 1.IaaS (Infrastructure as a Service)

•Provides virtual servers, storage, and networking.

### User manages:
•OS

### Applications
•Provider manages:
•Physical hardware
### Example
•Rent Infrastructure

### 2.PaaS (Platform as a Service)

•Provider manages infrastructure + OS.

•User focuses on developing and deploying applications.

### Example

•Build & Deploy Apps

### 3.SaaS (Software as a Service)

•Complete software delivered over the internet.

•Provider manages everything.

### Examples

•Gmail

•Zoom

# Major Cloud Vendors
There are several cloud vendors offering a variety of services, but Amazon Web Services (AWS) is the industry leader, with the most extensive offerings and global reach. Other well-known cloud providers include:

•Microsoft Azure: A strong competitor, especially in enterprise and hybrid cloud environments.

•Google Cloud Platform (GCP): Known for powerful data analytics, AI, and machine learning tools.

•Alibaba Cloud: A major player in Asia, offering competitive cloud services globally.

•IBM Cloud: Focuses on hybrid cloud and AI-driven solutions for businesses.

•Oracle Cloud: Focuses on enterprise applications and databases.
