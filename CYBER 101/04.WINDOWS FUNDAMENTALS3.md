# **Windows Update – Important Notes**

* **Purpose:** Provides **security updates, bug fixes, feature updates, and patches** for Windows and Microsoft products.
* **Patch Tuesday:** Updates are usually released on the **2nd Tuesday of every month**.
* **Critical Updates:** Released immediately if urgent (don't wait for Patch Tuesday).
* **Location:** **Settings → Windows Update**
* **Run Command:** `control /name Microsoft.WindowsUpdate`

-------------------

# **Windows Security – Important Notes**

### **Protection Areas**

* **Virus & Threat Protection** – Protects against malware and viruses.
* **Firewall & Network Protection** – Manages firewall and network security.
* **App & Browser Control** – Protects against malicious apps, websites, and downloads.
* **Device Security** – Shows hardware-based security features (e.g., Secure Boot, TPM).

### **Status Icons**

* 🟢 **Green:** Device is protected; no action needed.
* 🟡 **Yellow:** Security recommendation available.
* 🔴 **Red:** Immediate attention required.

# **Virus & Threat Protection – Important Notes**

#### **1. Current Threats**

**Scan Options:**

* **Quick Scan:** Scans common locations for malware.
* **Full Scan:** Scans the entire system (takes longer).
* **Custom Scan:** Scans selected files or folders.

**Threat History:**

* **Last Scan:** Shows the latest antivirus scan.
* **Quarantined Threats:** Isolates detected threats to prevent execution.
* **Allowed Threats:** Threats manually permitted to run (**use with caution**).

* Manage Microsoft Defender settings, including **Real-time Protection**, **Cloud-delivered Protection**, and other security options.

# **Windows Defender Firewall – Important Notes**

* **Purpose:** Monitors and controls incoming and outgoing network traffic through ports.
* **Analogy:** Acts like a **security guard**, allowing or blocking network connections.

### **Firewall Profiles**

* **Domain:** Used when connected to an organization's domain network.
* **Private:** For trusted networks (Home/Lab).
* **Public:** For untrusted networks (Public Wi-Fi, airports, cafes).

### **Firewall Options**

* Turn **Firewall On/Off**.
* **Block all incoming connections**.

### **Command**

* **Open Windows Defender Firewall:** `wf.msc`

---------------------

# **App & Browser Control – Important Notes**

* **Purpose:** Protects against **phishing websites, malicious apps, and unsafe downloads** using **Microsoft Defender SmartScreen**.

### **Features**

* **Microsoft Defender SmartScreen:** Warns or blocks malicious websites, apps, and downloads.
* **Check Apps & Files:** Scans unrecognized apps/files from the internet before running them.
* **Exploit Protection:** Built-in security that protects Windows from common software exploits.

### **SmartScreen Modes**

* **Warn** – Alerts before opening risky content (default).
* **Block** – Blocks dangerous apps/websites.
* **Off** – Disables SmartScreen protection (not recommended).

# **Device Security (Windows Security) – Short Notes**

* **Core Isolation:** Protects critical system processes from attacks.
* **Memory Integrity:** Blocks malicious code from entering high-security processes.
* **Security Processor (TPM):** Hardware chip that securely performs cryptographic operations and stores security keys.
* **TPM (Trusted Platform Module):** Tamper-resistant security chip that provides hardware-based protection for encryption, authentication, and system security.
* **Note:** Keep default security settings unless you know exactly what you're changing.

------------

# **BitLocker (Windows Security) – Short Notes**

* **BitLocker:** Full disk encryption feature that protects data from theft or unauthorized access.
* **Uses TPM (Trusted Platform Module):** TPM securely stores encryption keys and verifies system integrity.
* **Best Protection:** BitLocker works best with **TPM 1.2 or later**.
* **Benefit:** Prevents data access if a device is **lost, stolen, or tampered with**.

--------------------------

# **Volume Shadow Copy Service (VSS) – Short Notes**

* **VSS (Volume Shadow Copy Service):** Creates **shadow copies (snapshots)** of data for backup and recovery.
* **Stored in:** `System Volume Information` folder on protected drives.
* **Functions:** Create Restore Point, System Restore, Configure Restore Settings, Delete Restore Points.
* **Security Note:** Malware/ransomware often deletes shadow copies to prevent data recovery.
* **Best Practice:** Always maintain **offline/off-site backups** in addition to VSS.
